Installation

Connect BASTION to your Claude client in under 60 seconds.

Add BASTION to your Claude Code MCP settings:

~/.claude/settings.json 
{
  "mcpServers": {
    "bastion": {
      "url": "https://bastionfi.tech/mcp/sse",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}

Restart Claude Code after saving. Your agent now has access to all 10 BASTION tools.

Add to your Claude Desktop configuration file:

claude_desktop_config.json 
{
  "mcpServers": {
    "bastion": {
      "transport": "sse",
      "url": "https://bastionfi.tech/mcp/sse",
      "headers": {
        "Authorization": "Bearer YOUR_API_KEY"
      }
    }
  }
}

macOS: ~/Library/Application Support/Claude/claude_desktop_config.json

Windows: %APPDATA%\Claude\claude_desktop_config.json

Connect from any MCP-compatible client:

SSE Connection 
# SSE Endpoint
https://bastionfi.tech/mcp/sse

# Headers
Authorization: Bearer YOUR_API_KEY

# Message Endpoint (POST)
https://bastionfi.tech/mcp/messages/?session_id={id}
Python (mcp SDK) 
from mcp import ClientSession
from mcp.client.sse import sse_client

async with sse_client(
    "https://bastionfi.tech/mcp/sse",
    headers={"Authorization": "Bearer YOUR_API_KEY"}
) as (read, write):
    async with ClientSession(read, write) as session:
        await session.initialize()
        tools = await session.list_tools()
        print(f"Connected: {len(tools.tools)} tools")

Tools Reference

10 tools available to your Claude agent. Each tool is called automatically when relevant to the conversation.

CORE

bastion_evaluate_risk

Evaluate a crypto futures position using BASTION's AI risk intelligence engine. Sends a position to the fine-tuned 32B model which analyzes 560+ real-time signals and returns a specific action with reasoning.

ParameterTypeRequiredDescription
symbolstringYesTrading pair (e.g. BTC, ETH, SOL)
directionstringYesLONG or SHORT
entry_pricefloatYesPrice at which position was opened
current_pricefloatYesCurrent market price
leveragefloatNoLeverage multiplier (default 1.0)
stop_lossfloatNoStop loss price (0 = no stop)
take_profit_1floatNoFirst take profit target
take_profit_2floatNoSecond take profit target
take_profit_3floatNoThird take profit target
position_size_usdfloatNoPosition size in USD (default 1000)
duration_hoursfloatNoHours position has been open
Example prompt 
"Evaluate my BTC long: entered at $95,000, current price $97,500,
10x leverage, stop at $93,000, TP1 at $100,000"
Returns: action (HOLD / EXIT_FULL / TP_PARTIAL / EXIT_100% / REDUCE_SIZE / TRAIL_STOP), urgency, confidence (0-1), reason, detailed reasoning, and execution details.

Get recent whale transactions across 11 blockchains. Tracks large transfers (>$1M) with exchange attribution.

ParameterTypeRequiredDescription

bastion_get_funding_rates

Get cross-exchange funding rates. Positive = longs paying shorts (bullish overcrowding), negative = shorts paying longs (bearish overcrowding).

No parameters required.

Example

API Keys

An API key is required to authenticate your Claude agent with BASTION MCP. Keys are passed via the Authorization header.

Getting a Key
  1. Create a BASTION account at bastionfi.tech
  2. Navigate to Account → API Keys
  3. Generate a new MCP API key
  4. Add the key to your Claude client config (see Installation)
Header format 
Authorization: Bearer bst_live_xxxxxxxxxxxxxxxxxxxx
